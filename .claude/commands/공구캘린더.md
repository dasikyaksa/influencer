인스타그램 공동구매 캘린더(`index.html`)를 생성하거나 수정합니다.

## 주요 기능
- Google Sheets API로 스프레드시트 데이터 실시간 연동
- 인스타그램 4:5 비율(1080x1350) 캘린더 자동 생성
- 제품명/설명/배지 인라인 편집
- 테마 색상 변경
- html2canvas로 PNG 이미지 다운로드

## 작업 시 참고사항

1. **파일**: `index.html` 단일 파일 (CSS + JS 포함, 파일 분리 금지)
2. **기술 스택**: 순수 HTML + CSS + JavaScript (프레임워크 없음)
3. **CSS 변수**: 테마 색상은 `--accent` (RGB), `--accent-hex` (HEX) 사용
4. **pseudo-element 금지**: html2canvas 호환성 문제로 실제 DOM 요소 사용
5. **API 키 하드코딩 금지**: localStorage에 저장, UI에서 입력

## 스프레드시트 컬럼 (A~H)
| 컬럼 | 내용 |
|------|------|
| A | 공구번호 |
| B | 공구타이틀 |
| C | 상품명 |
| D | 공구시작일시 |
| E | 공구종료일시 |
| F | 판매링크 URL |
| G | 공구 상태코드 |
| H | 썸네일이미지 URL |

## 사용자 요청에 따라

- **새 제품 추가**: `getShortName()`, `getProductDesc()`, `getBadge()` 매핑 수정
- **디자인 변경**: CSS 변수 및 스타일 수정
- **기능 추가/수정**: JavaScript 로직 수정
- **버그 수정**: 관련 코드 확인 후 수정

수정 완료 후 사용자에게 로컬 테스트(`npx serve .`) 또는 배포(`/deploy`) 안내를 제공하세요.
