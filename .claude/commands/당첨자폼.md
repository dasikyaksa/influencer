이벤트 당첨자 배송지 입력 폼을 생성하거나 수정합니다.

## 개요
인스타그램 이벤트(공동구매·증정) 당첨자가 배송지를 제출하고, 운영자가 Google Sheets에서 응답을 관리하는 정적 HTML 웹 애플리케이션입니다.

## 파일 구조
| 파일 | 역할 |
|------|------|
| `event.html` | 당첨자 배송지 입력 폼 |
| `event-admin.html` | 응답 조회·삭제·내보내기 관리자 페이지 |
| `Code.gs` | Google Apps Script 백엔드 (참조용) |

## 상세 스펙
**반드시 `event-prompt.md` 파일을 읽고 참고하세요.** 이 파일에 디자인, 기능, API, 배포에 대한 전체 명세가 있습니다.

## 작업 시 참고사항

1. **단일 파일**: 각 HTML 파일에 CSS + JS 모두 포함 (파일 분리 금지)
2. **프레임워크 없음**: 순수 HTML + CSS + JavaScript
3. **의존성 (CDN)**:
   - 폰트: Pretendard
   - 주소 검색: Daum Postcode API (event.html만)
   - 엑셀 내보내기: SheetJS XLSX (event-admin.html만)
4. **Google Apps Script 연동**: Apps Script URL은 상수로 관리
5. **CORS**: POST 시 Content-Type 헤더 없이 전송 (preflight 회피)
6. **오렌지 테마**: `--brand-primary: #FF6B35` 기준 그라데이션

## 사용자 요청에 따라

- **새 이벤트 폼 생성**: `event-prompt.md` 스펙 기반으로 `event.html` + `event-admin.html` 생성
- **제품 정보 변경**: 상품 카드, 제품명 기본값, 타이틀 등 수정
- **폼 필드 추가/제거**: 폼 구조 및 검증 로직 수정
- **디자인 변경**: CSS 변수 및 스타일 수정
- **Apps Script 설정**: Code.gs 코드 제공 및 배포 안내

| 컬럼 | 내용 |
|------|------|
| 제출번호 | 자동 발급 (서버) |

| 인스타그램 | 사용자 핸들 |
